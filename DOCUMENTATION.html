Quick Start

  1. Extract the project files
  2. Set up MySQL database: mysql -u root -p < backend/db/schema.sql
  3. Configure backend: cd backend && cp .env.example .env (edit .env with your DB credentials)
  4. Install & start backend: npm install && npm run dev
  5. In another terminal, start frontend: cd frontend && npm install && npm start
  6. Access app at http://localhost:3000

Troubleshooting

Database Connection Error

Error: connect ECONNREFUSED 127.0.0.1:3306

  • Ensure MySQL is running
  • Verify DB credentials in .env
  • Run: mysql -u root -p to test connection

Port Already in Use

Error: listen EADDRINUSE :::5000

  • Change PORT in .env
  • Or kill process: lsof -ti:5000 | xargs kill -9

Frontend Can't Connect to Backend

Failed to fetch from localhost:5000

  • Ensure backend server is running
  • Check CORS is enabled in server.js
  • Verify API URLs in components
